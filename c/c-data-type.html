常量与变量

常量

常量（Constant）：在程序中不能改变其值的量

变量

变量(Variable)：其值在程序执行过程中可以改变

变量命名规则
  • 标识符只能由英文字母、数字和下划线组成；
  • 标识符必须以字母或下划线开头；
  • 不允许使用c关键字为标识符命名；
  • 标识符可以包含任意多个字符，但有最大长度限制，与编译器相关。

注 ：变量名要有实际意义。

变量声明

类型关键字+变量名;

使用变量的基本原则：

  • 变量必须先声明，后使用
  • 所有变量必须在第一条可执行语句前声明
变量的赋值

  1. 声明变量的同时为变量赋值称为变量的初始化；不被初始化的变量，其值为随机数（乱码）

  2. 通过赋值表达式语句进行赋值

规则：
变量←表达式 左值和右值类型一致

变量属性：变量的值、变量的名、变量的地址、变量被分配的内存大小

  • 变量的类型（Type）：决定变量被分配内存空间的大小

    内存是指计算机的存储部件（特点：所有指令和数据都保存在内存里，速度快，可随机访问，但掉电即失）

    内存编址：①地址按字节编号，内存中每一个字节（Byte,8个位表示一个字节，即1B=8b)都有唯一的编号；

    ​ ②地址是一个十六进制无符号整数，字长与主机相同，32 位计算机的内存地址编码是32位，从0×00000000到0×FFFFFFFF;

    ③按变量声明的类型分配内存空间大小类似于博士生、研究生、本科生分配寝室

数据类型

常见数据类型

  • 基本类型：

    • 整型：基本整型（int）、长整型（long）、短整型（short）、无符号整型（unsigned）
    • 实型（浮点型）：单精度实型（float）、双精度实型（double）、长双精度实型（long double）
    • 字符型（char）
    • 枚举类型（enum）

  • 构造类型：

    • 数组（-）
    • 结构体（struct）
    • 共同体（union）

  • 指针类型

  • 无类型
占用内存空间大小

计算机存储数据的最基本单元，衡量物理存储器容量的最小单位——位，也称比特（bit，binary digit二进制的缩写，其值只能是0或1）；8位（bit）可表示0~255间的整数，即一个字节（Byte，最小的可寻址的存储期单位，也是标识字符的最小单位）

1B=8b 1KB=1024B 1MB=1024KB 1GB=1024MB 1TB=1024GB

变量的类型决定占用内存空间的大小、数据的储存形式、合法的取值范围、可参与的运算种类

  • int，基本整型，c标准为规定，系统相关(大多数系统占4个字节)

  • short，短整型，2个字节

  • long int,长整型，4个字节

  • unsigned,无符号整型，用来修饰int,short和long（正整数和零）

  • float,单精度实型，4个字节

  • double，双精度实型，8个字节

  • long double，长精度实型，系统相关（VC++上占8个字节）

  • char，字符型，1个字节

数据类型 占用大小 表示范围
int 4字节 {-21474833648,214783647}
short 2字节 {-32768,32767}

注释

通常用于对程序中的某一行代码进行解释

单行注释

用”//“符号表示，后面为注释内容

多行注释

注释中的代码可以为多行,用符号/ /表示

注：在c语言中，有的注释可以嵌套使用，有些不能。

多行注释”/ /“中可以嵌套单行注释

多行注释中不能嵌套多行注释

屏幕输出

printf()格式

  • printf(“字符串\n”)
  • printf(“输出控制符”，输出参数)；

注：这句话的意思是将变量a以十进制输出

  • printf(“输出控制符1输出控制符2…”，输出参数1，输出参数2，…)；
  • printf(“输出控制符 非控制符”，输出参数)；

sizeof()

定义：一个操作符(operator).

其作用是返回一个对象或所占的类型所占的内存字节数

语法

  • sizeof(object); //sizeof(对象)
  • sizeof object ; //sizeof对象
  • sizeof(type)

对象可以是各种类型的变量，以及表达式（一般sizeof 不会对表达式进行计算）。sizeof对对象求内存大小，最终都是转化为对对象的数据类型进行求值。

sizeof（表达式）；//值为表达式的最终结果的数据类型的大小
