算法的概念及其描述方法

数据结构 + 算法 = 程序

  • 算法：解决一个具体问题而采取的确定、有限、有序、可执行的操作步骤。（其中某些步骤的顺序是不能改变的）
  • 数据结构（Date Structure)：是计算机存储、组织数据的方式，指相互之间存在一种或多种特定关系的数据元素的集合）

关系运算符与关系表达式

简单的程序基本操作：

输入所需的数据 –> 进行运算和数据处理 –> 输出运算结果

（这是一种常见的程序结构，即顺序结构。赋值和数据的输入/输出是顺序结构中最典型的操作，主要由表达式语句组成。表达式语句由表达式后接一个分号组成）

关系运算符
符号 意义 举例 结果
> 大于 10>5 1
>= 大于等于 10>=10 1
< 小于 10<5 0
<= 小于等于 10<=10 1
== 等于 10==5 0
!= 不等于 10!=5 1

注意：>=,<=,==,!=这种符号之间不能存在空格

关系表达式

用关系运算符将两个操作数连接起来组成的表达式。通常用于表达一个判断条件，结果只有两种可能：“真”或“假”

//只要表达式的值为0，则表达式的值为假，即判断不成立；若表达式的值为非0值（包括负数），则表达式的值为真，即表达式所表示的判断条件成立

逻辑运算符
符号 意义 举例 结果
&& 与 0&&1 0
|| 或 0||1 1
! 非 !0 1

逻辑运算的值也是有两种分别为真和假，C语言中用整型1和0表示

求值规则如下：

与运算&&

参与运算的两个变量都为真，结果才为真，否则为假。

或运算||

参与运算的两个变量只要有一个为真，结果就为真；两个变量都为假。

注：运算符&&和||都具有“短路”特性

//要使a && b运算结果为真则要求：a为假，则不计算b；

要使a || b运算结果为假则要求：a为真，则不计算b

非运算!

参与运算的变量为真时，结果为假；参与运算量为假时，结果为真。
