vim模式：

编辑模式（命令模式）
输入模式
末行模式

模式转换：
编辑–>输入：
i：在当前光标所在字符的前面，转为输入模式
a：在当前光标所在字符的后面，转为输入模式
o：在当前光标所在行的下方，新建一行，转为输入模式
I：在当前光标所在行的行首，转为输入模式
A：在当前光标所在行的行尾，转为输入模式
O：在当前光标所在行的上方，新建一行，转为输入模式

输入–>编辑：
ESC

输入–>末行：
:

末行–>编辑：
ESC，ESC

一、打开文件

vim FILE
vim +#：打开文件直接定位到第#行
vim +：打开文件，定位至最后一行
vim +/PATTEN：打开文件，定位至第一次被PATTEN匹配到的行首

默认处于编辑模式

二、关闭文件

1、末行模式下关闭文件
:q 退出
:wq 保存并退出
:q! 不保存退出
:w 保存
:w! 强行保存
:wq –> :x
2、编辑模式下退出
ZZ：保存并退出

###三、移动光标
1、逐字符移动
h：左
l：右
j：下
k：上

#h：移动#个字符

2、逐单词移动
w：移至下一个单词词首
e：跳至当前或下一个单词词尾
b：跳至当前或前一个单词词首
3、行内跳转
0：绝对行首
^：行首的第一个非空白字符
$：绝对行尾
4、行间跳转

#G：跳转至第#行

G：最后一行
末行模式下，直接给出行号即可

四、翻屏

Ctrl+f：向下翻
Ctrl+b：向上翻

Ctrl+d：向下翻半屏
Ctrl+u：向上翻半屏

五、编辑

x：删除光标所在的单个字符

#x：删除光标所在及其向后共#个字符

六、删除命令：d

d$、 d0、#dw、#de、#db
dd：删除当前光标所在行

#dd：删除包括当前所在行在内的#行

末行模式下：
StartADD,EndADD d：删除从start到end行
.：表示当前行
$：最后一行
+#：向下的#行

七、粘贴命令 p

p：如果删除或复制的为整行，则粘贴至光标所在行的下方，如果复制 或删除的内容为非整行，则粘贴至光标所在字符的后面
P：如果删除或复制的为整行，则粘贴至光标所在行的上方，如果复制 或删除的内容为非整行，则粘贴至光标所在字符的前面

八、复制 y（yank）

用法同d命令

九、修改：先删除内容再转换为输入模式

c：用法同d

十、替换：

r：替换单个字符
R：替换模式

十一：撤销编辑

u：（undo）撤销前一次操作，可多次使用

#u：直接撤销#次编辑操作

撤销最近一次撤销：Ctrl+r

十二、重复前一次编辑操作

.

十三、可视化模式

v：按字符选取
V：按矩形块选取

十四、查找

/PATTERN
n：下一个
N：上一个
?PATTERN：倒着找

十五、查找并替换

在末行模式下使用s命令
ADDR1,ADDR2@PATTERN@string@gi：
g：全局替换
i：忽略大小写
例如 :.,$s/aa/AA/g：表示从当前行，到末行把aa替换为AA，全局替换
%：表示全文 :%s/aa/AA/g

十六、使用vim编辑多个文件

vim FILE1 FILE2 FILE3
:next：切换至下一个文件
:prev
:last
:first

退出
:qa 全部退出

十七、分屏显示一个文件

Ctrl+w, s：水平拆分窗口
Ctrl+w,v：垂直拆分

在窗口间切换光标：
Ctrl+w，↑↓←→

十八、分窗口显示多个文件

vim -o：水平分割 显示
vim -O：垂直分割显示

十九、将当前文中部分内容另存为另外 一个文件

末行模式下使用w命令
:w
:w /path

二十、将另一个文件的内容填充到当前文件

:r

二十一、和shell交互

:! COMMAND

二十二、高级话题

1.显示行号
:set nu：number 行号

:set nonu

2.显示忽略大小写
:set ic（ignorecase）

:set noic（noignorecase）

3.设定自动缩进
:set autoindent
:set ai

:set noai

4.查找到的文本高亮显示或取消
:set hlsearch
:set nohlsearch

5.语法高亮
:syntax on
:syntax off
